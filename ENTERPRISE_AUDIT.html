6. Compliance & Finance

  • P0 e-Invoice (IRN + signed QR) — mandatory above turnover thresholds; integrate a GSP/IRP API; print IRN+QR on invoice.
  • P0 e-Way Bill — auto-generate for dispatch > ₹50,000; transporter/vehicle, Part-A/B; cancel/extend.
  • P1 GST returns — GSTR-1 & 3B data export (JSON/Excel); HSN summary; B2B/B2C split. (TCS u/s 206C(1H) is repealed Apr-2025 — keep out.)
  • P1 Credit/Debit notes — for returns, rate diff, short supply; linked to invoice; GST-compliant.
  • P1 Tally/Busy integration — export vouchers (sales, receipts, purchases) so accountants stop double-entering.
  • P2 Bank reconciliation — import statement, match payments, flag unreconciled.

9. Prioritised Roadmap

🚀 NOW (P0 — foundation & compliance, do first)

  • Audit trail (every change logged) + soft-delete/restore.
  • RBAC: granular permissions + cost/margin field masking (sales vs admin).
  • e-Invoice (IRN/QR) + e-Way Bill auto-generation.
  • Material availability check before run scheduling + low-stock → auto-PO suggestion.
  • Credit-limit block + WhatsApp/UPI payment reminders (cut receivables).
  • Security: 2FA, rate-limiting, encrypt sensitive fields; automated backups + staging.

⏭ NEXT (P1 — depth & efficiency)

  • Approval workflows (discount/PO/credit/write-off).
  • Returns + Credit/Debit notes; Tally export.
  • Production capacity/shift scheduling (Gantt) + cutting-plan optimiser + yield analytics.
  • WhatsApp + in-app notification centre; web lead-capture form.
  • Profitability/margin reporting; data import/export; KPI dashboards with targets.
  • Shop-floor mobile/PWA + barcode/QR traceability.

🔮 LATER (P2 — scale & productise)

  • Multi-warehouse/plant + transfers; lot traceability.
  • Dealer & customer portals; vendor portal + 3-way match.
  • Transporter management + ePOD + live delivery tracking.
  • Machine maintenance/OEE; QC defect/CAPA + certificates.
  • SaaS: onboarding wizard, plans/billing, super-admin console, marketplace integrations.
  • Bank reconciliation; global search & saved views; report builder.

Suggested sequencing principle: First make it trustworthy (audit, RBAC, security, backups) and compliant (e-invoice/e-way), then efficient (scheduling, returns, WhatsApp, reporting), then scalable/sellable (multi-plant, portals, SaaS). Each item is independent and additive — exactly how we've been shipping.
